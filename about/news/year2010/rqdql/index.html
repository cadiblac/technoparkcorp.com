    Formal And Easy To Understand Software Requirements

    Read more on this topic at RQDQL website.

    TechnoPark Corp. started a new initiative, called Requirements Definition and Query Language (RQDQL), which is going to design a method and software for automated specification of software requirements in formal way at the same time easy to understand for non-professional reader (project sponsor).

    The project is still in development and preliminary results will be published before the end of this year.
